Follow these step to install MedesUI Library on your project.

  1. Make sure node.js installed on your computer.
  2. Select or cd project want to instal the library. Run on terminal/cmd npm install medes-ui.
    If doesn't work that mean UI itself is using medes-ui beta version module. Copy dist/medes-ui folder on this repository to node_modules folder.
  3. Import medes-ui Stylesheet by following these steps:
    • Import medes-ui.scss stylesheet on main styles.scss.
    • Or import to angular.json.
    Documentation For creating your own color scheme click here.
  4. Import modules, components or utils to use.
  5. Run with ng serve or npm start

Another Dependencies required to install

  • Install PrismJs, npm i prismj and import stylesheet by following step similar with importing medes-ui.scss. More info visi official site here.
