Follow these step to install MedesUI Library on your project.

  1. Make sure node.js installed on your computer.
  2. Select or cd project want to instal the library. Run on terminal/cmd npm install medes-ui, Or if you want to with spesific version run npm i medes-ui@0.0.1.
  3. Import medes-ui Stylesheet by following these step
    • Import medes-ui Stylesheet on main styles.scss.
    • Or import to angular.json
    Documentation For creating your own color scheme click here.
  4. Import modules, components or utils to use.

Another Dependencies required to install

  • Install PrismJs npm i prismj and import style.
